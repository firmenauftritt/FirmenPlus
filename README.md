# FirmenPlus - Premium Digital Agency Website

## Overview

A modern, production-ready website for FirmenPlus, a premium digital agency serving small and medium-sized businesses in Germany.

## Tech Stack

- **Framework**: Next.js 15
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Deployment**: Ready for Vercel

## Features

✅ Full responsive design (mobile-first)
✅ SEO optimized with Next.js metadata
✅ Modern animations and transitions
✅ Professional UI/UX design
✅ Dark blue and white color scheme
✅ Contact form with validation
✅ FAQ accordion component
✅ Process visualization
✅ Services showcase
✅ Trust indicators
✅ Production-ready code
✅ Clean component structure
✅ TypeScript for type safety

## Pages

- **Home** (`/`) - Landing page with all sections
- **Contact** (`/contact`) - Contact form and information

## Sections on Home Page

1. **Navigation** - Sticky header with mobile menu
2. **Hero** - Eye-catching headline and CTAs
3. **Services** - 5 core service offerings
4. **Why Choose FirmenPlus** - 6 key benefits
5. **Process** - 4-step growth process
6. **FAQ** - Expandable Q&A section
7. **Contact CTA** - Strong call-to-action
8. **Footer** - Links and contact information

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view in browser.

## Build

```bash
npm run build
npm start
```
